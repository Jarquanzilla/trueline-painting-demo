# Trueline Painting Co. — demo site

A portfolio demo for a residential/commercial painting contractor. Bilingual (EN/ES),
responsive, built with Vite + Tailwind CSS. All business details, testimonials, and
imagery are fictional/placeholder for demonstration purposes.

## Stack
- Vite (static multi-page build)
- Tailwind CSS
- Vanilla JS (splash, mobile nav, scroll reveal, language toggle, mailto contact form)

## Develop
```bash
npm install
npm run dev      # local dev server
npm run build    # production build → dist/
npm run preview  # preview the build
```

## Notes
- Imagery in `images/` is AI-generated and used for demonstration only.
- Contact details (phone, address, email) are placeholders.
